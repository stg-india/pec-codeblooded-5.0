# H-1B Disclosure Data Conversion

This Python script converts data from an Excel file to a CSV file. It's designed for use with H-1B Disclosure Data.

## Prerequisites

Before running the script, ensure you have the required Python libraries installed. You can install them using pip:

```bash
pip install openpyxl


## Usage

1. Place your Excel file (e.g., 'H-1B_Disclosure_Data_FY16.xlsx') in the same directory as this script.

2. Run the script using Python:

bash
python excel_to_csv.py


3. The script will create a CSV file named 'H-1B_Disclosure_Data_FY16.csv' with the converted data.
